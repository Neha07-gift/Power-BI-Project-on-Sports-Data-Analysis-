**Project Overview:**

The Ultimate Sports Analytics Dashboard is an interactive Power BI report designed to analyze player, team, and match performance across multiple sports. It provides comprehensive insights through KPIs, charts, maps, and slicers, enabling users to explore trends, compare performances, and make data-driven decisions.
This project demonstrates Power BI skills including data modeling, DAX calculations, data visualization, and dashboard design.

**Objective:**

Build a comprehensive sports analytics dashboard that helps users:

Monitor overall match performance
Identify top-performing players
Compare sports based on scoring
Analyze monthly scoring trends
Visualize global player distribution
Filter reports dynamically using slicers

**Dashboard Features:**
1. KPI Cards

The dashboard includes four key performance indicators:

Total Matches Played
Total Points/Goals Scored
Average Points per Match
Player with the Most Points

2. Bar Charts
Top 5 Players by Total Points/Goals
Displays the highest-scoring players across all sports.
Points Scored by Sport
Compares total points scored in:
Football
Cricket
Basketball
Top 3 Players by Minutes Played
Highlights players with the highest playing time.

3. Pie Chart

Win / Loss / Draw Ratio
Shows the overall match outcomes for all teams, providing quick insight into team performance distribution.

4. Line Chart

Monthly Trend of Total Points Scored
Visualizes scoring performance over time to identify seasonal or monthly trends.

5. Map Visualization

Player Distribution by Country
Displays player locations on a world map to understand country-wise participation.

6. Interactive Slicers

Users can filter the dashboard using:

Player Name
Team
Sport
Country
Match Date

These slicers update every visualization dynamically for detailed analysis.
