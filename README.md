**# Power-BI-Project-on-Sports-Data-Analysis-**

**Objective**
The primary objective of this project is to provide meaningful insights into sports data by visualizing important metrics such as total matches played, total points scored, player performance, match trends, and geographical player distribution.   

**Project Overview**
The Ultimate Sports Analytics Dashboard is an interactive Power BI dashboard developed to analyze player performance, team statistics, match outcomes, and country-wise participation across multiple sports. The dashboard enables users to monitor key performance indicators (KPIs), compare player performance, identify trends, and interactively filter data using slicers.

**Dataset**
The dataset contains information related to sports matches and includes the following attributes:

Player Name
Team
Sport
Country
Match Date
Match Result
Points/Goals
Minutes Played
Assists
Position

**Dashboard Features**
1. KPI Cards:

The dashboard contains four KPI cards to summarize important statistics:

Total Matches Played
Total Points/Goals Scored
Average Points per Match
Player with the Most Points

These KPIs provide a quick overview of the overall dataset.

2. Bar Charts:

The dashboard includes multiple bar charts for performance comparison:

Top 5 Players by Total Points/Goals
Points Scored by Sport (Football, Cricket, Basketball)
Top 3 Players by Minutes Played

These visuals help identify top-performing players and sports.

3. Pie Chart:

A pie chart displays the distribution of:

Win
Loss
Draw

This visualization provides insights into overall match outcomes.

4. Line Chart:

The monthly trend analysis visualizes:

Total Points Scored by Month

This helps identify seasonal performance trends and scoring patterns over time.

5. Map Visualization:

The map chart displays:

Player Distribution by Country

This provides a geographical overview of player participation across different countries.

Note: If the default Map visual is unavailable, Azure Maps can be used as an alternative.

6. Interactive Slicers:

Users can dynamically filter the dashboard using:

Player Name
Team
Sport
Country
Match Date

These slicers make the dashboard interactive and allow customized analysis.
